# simba-plugins-sgamise
Dépôt des plugins développés par la Team SIMBA

- ActionLogPrune : Tâche de suppression des entrées dans le journal des actions avec rétention configurable.
- AkeebaCleanup : Contrôle de la chaine de sauvegarde et suppression en 1 clic des backups obsolètes réalisés par Akeeba Backup.
- ComposantsDSFR : Permet d'insérer des composants DSFR dans l'éditeur d'articles.
- DSFRChart : Permet d'insérer des graphiques basés sur le projet ChartJS du DSFR dans l'éditeur d'articles.
- ImageOptimiseur : Optimisation (compression et redimensionnement) des images à la volée au moment de l'upload dans le Média Manager.
- MediaUsage : Tableau de bord étendu de l'utilisation du stockage par les fichiers dans le Media Manager (top 50 des fichiers volumineux, recherche de fichiers orphelins,...).
- ModuleExport : Outil d'aide à la migration de template.
- ProtectMedia : Protection à la suppression et à l'upload sur des dossiers et fichiers du Media Manager.
- SoumissionsCF : Permet aux simples rédacteurs d'alimenter des soumissions à un formulaire ConvertForm.
